#### Prerequisite: 

1. virtualenv.
2. python 3.
3. Bash.

### Installation:

Production build Using docker-compose [nginx, uwsgi, postgresql, redis, django]:

       # clone the repo
       $ git clone git@github.com:misraX/strmatch.git
       $ cd strmatch
       # Run the build script.
       $ ./start.sh
